# 🎨 AI Product Image Generator
An end-to-end Generative AI application that transforms text descriptions into high-quality product photography using Stable Diffusion and Gradio.

## 🚀 Live Demo & Code
* **Kaggle Notebook:** [Run the code here](https://www.kaggle.com/code/abhinayvanjari/ai-product-image-generator-fv)
* **Built With:** Python, Gradio, Hugging Face Diffusers

## 🛠️ Key Features
- **Instant Generation:** Converts text prompts into studio-quality images in seconds.
- **Interactive UI:** A clean, user-friendly interface built with Gradio.
- **Optimized Prompts:** Pre-configured settings for professional lighting and product focus.

## 💻 Technical Stack
- **Library:** `diffusers` (Stable Diffusion)
- **Framework:** `gradio` (Interface)
- **Compute:** Optimized for CUDA/GPU environments.

## 📸 How to Use
1. Open the Kaggle link above.
2. Click **Copy & Edit**.
3. Ensure the GPU Accelerator is turned ON.
4. Run all cells to generate the public `gradio.live` link.
